记一次 Android Studio 更新

发表于 2017-08-09 | 分类于 Android | 阅读

3.0 Canary(金丝雀版)发布

据官方介绍，Android Studio 3.0包含了三大主要功能：一套全新的应用性能分析工具，用于快速诊断性能问题；支持Kotlin编程语言；加快大规模应用项目的 Gradle 构建速度。

在这之前问过一些网友，更多的反馈是说bug多，兼容性不是太理想，
从Canary2 到 3 和 4 不同版本间的工程都不能简单导入到新版本环境直接成功编译，追求稳定的可以静等正式版的发布。

有关更多详细信息，请查看谷歌开发者中文博客。

$FROM AI-162.3764568 就是版本号

  • 获取最新Android Studio的版本信息

可用的更新版本$TO AI-162.4069837

patch列表里面要能够找得到你当前所用的AS版本, 确保服务器有提供此差异包的下载

  • 下载补丁包

在浏览器中访问 https://dl.google.com/android/studio/patches/AI-162.3764568-162.4069837-patch-win.jar

  • 安装
  1. 将下载好的AI-162.3764568-162.4069837-patch-win.jar文件存放到非Android Studio的安装目录，这边放D盘根目录。
  2. 关闭AS，打开命令行工具进入到你当前AS的安装目录，然后执行 java -classpath D:/AI-162.3764568-162.4069837-patch-win.jar com.intellij.updater.Runner install .

. 表示安装到当前目录
